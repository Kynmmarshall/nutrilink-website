# NutriLink Website

Landing page for NutriLink, a community food redistribution platform. The site highlights the product vision, role-based journeys, impact roadmap, and app screenshots in a single-scroll experience.

## Highlights
- Hero + value proposition sections with CTA buttons.
- Feature cards, role journeys, community map, and impact roadmap.
- Screenshot gallery and developer profile section.
- Animated reveal on scroll and mobile-friendly navigation.

## Tech Stack
- HTML5
- CSS3 (custom properties, responsive grid)
- Vanilla JavaScript (navigation + scroll animations)
- Google Fonts: Space Grotesk, Playfair Display

## Project Structure
- index.html - Page markup
- styles.css - Styling and layout
- script.js - Navigation + scroll animations
- assets/ - Logo, screenshots, and team photos

## Run Locally
Open index.html in a browser. No build tools required.
